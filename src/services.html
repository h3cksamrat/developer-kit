    Setup SSH

    Generate SSH to connect your device with Github.

    When working with a GitHub repository, you'll often need to identify yourself to GitHub using your username and password. An SSH key is an alternate way to identify yourself that doesn't require you to enter you username and password every time.

    Clone Repository

    Clone any git repository within a fraction of second.

    You can clone a repository from Github to your local computer to make it easier to fix merge conflicts, add or remove files, and push larger commits.

    Commit

    Commit you git repository within a single click.

    The git commit command captures a snapshot of the project's currently staged changes. Committed snapshots can be thought of as “safe” versions of a project.

    Push Repository

    Push any git repository within a single click.

    The git push command is used to upload local repository content to a remote repository. Pushing is how you transfer commits from your local repository to a remote repo.

    Pull Repository

    Pull any git branch within a single click.

    The git pull command is used to fetch and download content from a remote repository and immediately update the local repository to match that content. Merging remote upstream changes into your local repository is a common task in Git-based collaboration work flows.
